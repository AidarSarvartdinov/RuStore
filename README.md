# RuStore – магазин приложений Android

Проект, имитирующий функциональность магазина приложений RuStore. 
Клиент написан на **Kotlin + Jetpack Compose**, бэкенд - на **Kotlin + Spring Boot**. 
Реализованы основные экраны: онбординг, витрина, категории, поиск, карточка приложения с возможностью фоновой загрузки приложений.

## Запуск проекта

### 1. Запуск сервера
- Откройте папку `server` как проект в IntelliJ IDEA.
- Запустите главный класс `RustoreApplication.kt` (или выполните `./gradlew bootRun` в терминале).
- Сервер поднимется на порту `8080`.


### 2. Запуск клиента
- Откройте папку `client` в **Android Studio**.
- Создайте виртуальное устройство с API 24+.
- Запустите эмулятор, выполните в терминале:
  ```
  adb reverse tcp:8080 tcp:8080
  ```
- Запустите приложение через кнопку **Run**


## Основные экраны и функции
- **Онбординг** – показывается при первом запуске.
- **Витрина приложений** – список приложений с иконками, кратким описанием и категорией.
- **Категории** – список категорий с количеством приложений; клик открывает витрину с фильтром.
- **Поиск** – текстовое поле с автопоиском по названиям; при пустом результате показываются популярные приложения.
- **Карточка приложения** – детальная информация: иконка, название, разработчик, категория, возрастной рейтинг, скриншоты, описание, кнопка "Установить".
- **Установка / Удаление** – фоновое скачивание APK-файла (с уведомлением о прогрессе) и запуск системного установщика. После установки кнопка меняется на "Открыть" и "Удалить".
- **Обработка ошибок** – при сетевых ошибках отображается экран с кнопкой "Повторить".
- **Pull-to-Refresh** – обновление контента на экранах со списками.

## Примечания
- Данные о приложениях (моки) загружаются с бэкенда при первом запуске.
- APK-файл является заглушкой.
